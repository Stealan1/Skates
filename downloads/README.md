## How to Download

You can download files directly from GitHub:
1. Navigate to the file in the GitHub repository
2. Click on the file name
3. Click the "Download" or "Raw" button
4. Save the file to your local machine

Or use direct links in your app:
```
https://raw.githubusercontent.com/Stealan1/Skatesmod/main/downloads/[filename]
```

Replace `[filename]` with the actual file name.
